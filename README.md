# 💧 Water Reminder (Python)

A simple Python automation script that reminds you to drink water at regular intervals using voice alerts.

## 🚀 Features
- Voice reminder using text-to-speech
- Automatic reminder after a fixed time interval
- Lightweight and easy to run

## 🛠 Tech Stack
- Python
- pyttsx3 (Text-to-Speech library)

## ▶️ How to Run

1. Install the required library:
pip install pyttsx3

2. Run the script:
python water_reminder.py

## ⚙️ How It Works
The script runs in a loop and reminds you to drink water after a fixed time interval using voice alerts.
